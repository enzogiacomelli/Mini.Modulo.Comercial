# Modulo.Comercial

Projeto criado para estudos.  

O projeto é uma ASP.NET Core Web API simples de gestão de pedidos(vendas), simulando um pequeno módulo de um sistema ERP.  


Cada pedido possui: 
```
Cliente
Lista de Produtos
Valor total
Status do Pedido
```

Tecnologias e frameworks:
```
C#
.NET 10
ASP.NET Core Web API
SQL (localDB)
Entity Framework
Front Simples(HTML/CSS e Angular básico)
```
